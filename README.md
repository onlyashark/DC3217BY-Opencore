# DC3217BY-Opencore

OpenCore 0.7.5 setup for running Hackintosh on Intel NUC DC3217BY

## Specifications:
- Intel Core i3 3217U (Ivy Bridge)
- Intel QS77 Chipset
- Intel HD 4000 Graphics
- 8GB DDR3 RAM
- 3 USB 2.0 Ports
- Thunderbolt 2
- HDMI Port
- 60G MSATA SSD
